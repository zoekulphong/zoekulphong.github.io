Crazy Ambulance

creating an interactive First aid game

My team and I created the Crazy Ambulance game and website in only 36 hours and won 1st place on the Health Tech Track at HackKU 2021!

introduction

Inspired by the Google dinosaur game and trivia games, Crazy Ambulance players drive an ambulance through streets with cars and roadblocks, and answer questions about how to handle medical situations while being reminded of the importance of calling 9-1-1. We created a website where the game and links to official First aid resources can be accessed.

This game was build in Unity, which uses C#. The game was broken down into three main sections: the basic game mechanics of play, pause, and replay, the game itself, and the quiz components of the game.

I created all of the graphics for Crazy Ambulance from scratch using an online pixel art creator, and programmed the quiz components of the game. Additionally, I designed the website where the game and additional First aid information can be found using Figma, and implemented it using HTML and CSS.
